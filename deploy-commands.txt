# Firebase Functions 배포 명령어

## 1. API 키 설정
# 발급받은 API 키를 아래 명령어의 YOUR_API_KEY 부분에 붙여넣으세요

firebase functions:config:set holiday.apikey="YOUR_API_KEY"

## 2. 설정 확인
firebase functions:config:get

## 3. Functions 배포
firebase deploy --only functions:getHolidays

## 4. 배포 확인
# 배포 후 이 URL로 테스트
# https://us-central1-adgd-bab.cloudfunctions.net/getHolidays
